# AI 健身动作纠错镜

> **一句话卖点**：健身新人在对着镜子瞎练时，用 AI 纠错镜实时识别关节角度，从深蹲腰酸背痛到每个动作都标准有效

## 概述

基于 Issue #199 创建的详细设计文档。

「对着镜子学动作」是健身新人的普遍困境——照着视频学深蹲，不知道膝盖有没有内扣；跟着 App 做平板撑，感觉不到腰有没有塌。请私教一节课 400 元太贵，不请又怕练错受伤。

本产品通过手机摄像头实时捕捉动作，AI 分析关节角度，用语音即时纠正：「膝盖内扣了，向外打开一点」。每个动作都有评分，让你知道「哪里做得对，哪里需要改进」。

**核心价值**：实时纠错 + 数据积累，比私教便宜，比自学安全。

## 功能设计

### 核心功能

1. **实时姿态分析**
   - 摄像头捕捉：手机/电视/镜子摄像头
   - 关节识别：17 个关键点（肩、肘、腕、髋、膝、踝等）
   - 角度计算：实时分析关节角度（如深蹲时膝盖-髋-脚尖连线）
   - 姿态评分：每个动作 0-100 分

2. **语音即时纠正**
   - 实时提示：「膝盖内扣了，向外打开一点」
   - 延迟控制：< 500ms（确保用户能及时调整）
   - 语气设计：鼓励式而非批评式
   - 本地 TTS：降低延迟，保护隐私

3. **动作计数 + 质量评分**
   - 智能计数：自动识别一个完整动作
   - 质量评分：每个动作的完成度（0-100%）
   - 综合评级：本次训练的整体评分

4. **常见错误库**
   - 深蹲：膝盖内扣、弓背、重心前移
   - 硬拉：弓背、起始位置不对、拉起轨迹错误
   - 卧推：手腕弯曲、落点不对、起桥过度
   - 俯卧撑：塌腰、翘臀、手肘角度不对
   - 平板撑：塌腰、翘臀、头部位置不对

5. **训练记录与进展追踪**
   - 动作历史：每个动作的质量变化曲线
   - 周期报告：本周训练次数、平均评分、进步幅度
   - 目标设定：如「深蹲评分达到 90 分」

6. **受伤风险预警**
   - 高危动作识别：如深蹲时膝盖超过脚尖过多
   - 累积风险提示：连续弓背可能伤腰
   - 休息建议：检测到疲劳迹象时提醒休息

## 技术方案

### 数据层
- **视频流处理**：MediaPipe Pose（17 关键点）
- **动作数据库**：标准动作角度库 + 常见错误库
- **用户数据**：本地 SQLite + 可选云端同步
- **隐私保护**：视频流本地处理，不上传云端

### 逻辑层
- **姿态估计**：MediaPipe BlazePose（轻量、实时）
- **动作识别**：基于关节角度序列的分类器
- **评分引擎**：标准动作对比 + 误差计算
- **语音生成**：本地 TTS（降低延迟）

### 展示层
- **前端**：React Native（跨平台）+ 实时渲染
- **摄像头**：手机前置/后置 + 可选电视投屏
- **语音**：本地 TTS + 外接音箱（可选）
- **可视化**：关节点叠加显示 + 角度标注

### 关键技术指标
- **识别精度**：关节点识别准确率 > 95%
- **延迟控制**：语音提示延迟 < 500ms
- **离线能力**：核心功能可离线使用
- **设备适配**：支持主流手机型号

## 实现步骤

### Phase 1：MVP（2 周）
- 单一动作：深蹲（最高频、最容易受伤）
- 基础姿态识别 + 语音提示
- 简单评分系统

### Phase 2：动作扩展（2 周）
- 扩展到 5 个高频动作（深蹲、俯卧撑、平板撑、硬拉、卧推）
- 完整错误库
- 动作计数

### Phase 3：数据追踪（2 周）
- 训练记录
- 进展曲线
- 周报生成

### Phase 4：高级功能（2 周）
- 受伤风险预警
- 多设备适配（电视/镜子）
- 社交分享（动作达标海报）

**Phase 1 就足够验证 PMF**：核心功能是「深蹲 → 识别 → 纠错 → 评分」。

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| 角度识别精度受光线/角度影响 | 多角度校准 + 光线检测提示 + 用户手动微调 |
| 侧面角度识别准确率低 | 引导用户调整手机位置 + 多角度采样 |
| 延迟过高影响体验 | 本地处理优先 + 优化算法 + 降级策略 |
| 不同设备摄像头角度不同 | 设备适配库 + 用户校准模式 |
| 用户隐私担忧（摄像头） | 本地处理 + 明确告知 + 数据不上传 |

## 差异化优势

| 现有方案 | 本产品 |
|---------|--------|
| Keep 等只计数，不评估质量 | 每个动作评分，告诉你哪里改进 |
| 私教 400/节，每月 1600+ | ¥19.9/月，24 小时随时用 |
| 视频教程只能看，不知道对不对 | 实时语音纠正，每个动作都有反馈 |
| 镜子只能自己看，看不出问题 | AI 分析关节角度，精确到度数 |
| 健身房镜子，只能在健身房用 | 手机随时用，居家/户外/健身房通用 |

## 市场规模

- **目标人群**：中国健身人群超 3 亿，私教渗透率 < 5%
- **核心痛点**：请私教太贵（300-500/节），自学怕受伤
- **付费意愿**：为「不受伤 + 练对动作」付费意愿强
- **市场空白**：现有 App 只有课程，没有实时纠错

## 用户场景

### 场景 1：居家健身新手
- **痛点**：跟着 B 站视频学深蹲，不知道做得对不对
- **使用**：手机靠在桌角，开始深蹲，AI 实时提示「腰背挺直」
- **效果**：从「练完腰酸」到「练完腿酸」（说明练对了）

### 场景 2：健身房常客
- **痛点**：私教课太贵，想自己练又怕动作退步
- **使用**：手机放在地面，录制侧面视角，AI 评分每个动作
- **效果**：保持动作标准，省下私教钱

### 场景 3：康复训练人群
- **痛点**：医生说「多做康复训练」，但不知道做得对不对
- **使用**：按照康复方案做动作，AI 监测关节角度
- **效果**：确保康复效果，避免二次受伤
