# AI 口语陪练伴侣 - 随时随地练习真实对话

> 让 AI 成为你的 24/7 外语陪练，场景化对话 + 实时纠音，从此告别「不敢开口」

## 功能设计

### 核心功能
1. **场景化对话练习** - 点餐、问路、面试、商务谈判等真实场景模拟
2. **实时语音纠错** - 发音、语调、用词即时反馈，不打断对话流
3. **自适应难度引擎** - 根据用户水平自动调整词汇复杂度和语速
4. **文化语境贴士** - 地道表达建议，避免文化误解
5. **学习进度追踪** - 可视化练习时长、纠错趋势、场景覆盖

### MVP 范围
- 3 个高频场景：餐厅点餐、问路导航、自我介绍
- 中英双语支持（后续扩展日/韩/西）
- Web 端 + 移动端响应式

## 技术方案

### 数据层
```
用户数据:
  - 语音录音 (OSS/S3)
  - 练习历史 (PostgreSQL)
  - 发音基线 (向量数据库)

场景数据:
  - 对话模板 (JSON Schema)
  - 文化知识库 (RAG)
  - 难度标签 (分级系统)
```

### 逻辑层
```
语音处理流水线:
  ASR (Whisper) → 意图识别 → 对话管理 → 响应生成 → TTS (Edge TTS)
       ↓
  发音评估 (发音准确度/流利度/完整度)
       ↓
  纠错建议生成 (LLM)
```

### 展示层
- Web: Next.js + Web Audio API
- Mobile: React Native 或 PWA
- 实时波形可视化

## 实现步骤

### Phase 1: MVP (4 周)

2. **Week 3-4**: 核心体验

   - 自适应难度算法

4. **Week 7**: 打磨
   - 性能优化

   - A/B 测试准备

### Phase 3: 商业化 (2 周)
5. **Week 8-9**: 变现

## 资源需求

### API & 服务
- **语音识别**: OpenAI Whisper API / 阿里云语音
- **语音合成**: Edge TTS (免费) / Azure TTS (高质量)
- **LLM**: Claude API / GPT-4o (对话 + 纠错生成)
- **发音评估**: 需自研或对接专业 SDK

### 基础设施
- **计算**: Vercel / Railway (Web), 云函数 (语音处理)
- **存储**: Supabase (数据库), OSS (录音)
- **监控**: Sentry + 自定义埋点

### 人力估算
- 1 全栈开发 (9 周)
- 0.5 产品设计 (3 周)
- 外教顾问 (按需)

## 风险与对策

| 风险 | 对策 |
|------|------|
| 用户担心「纠错太多」 | 正向反馈优先，纠错比例可调 |
| 付费转化不确定 | 先 MVP 免费积累用户 |
| 发音评估技术门槛 | 可对接成熟 SDK 降低风险 |

- **用户**: 7 日留存 > 30%
- **参与度**: 人均周练习时长 > 30 分钟
- **变现**: 付费转化率 > 5%
