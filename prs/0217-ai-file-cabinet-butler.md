# AI 文件柜管家

> **一句话卖点**：纸质文档囤积族拍照即归档，AI 自动识别分类，从翻箱倒柜 2 小时到 3 秒搜出任何文件。

## 概述

基于 [Issue #217](https://github.com/ava-agent/ai-ideas/issues/217) 创建。

家庭纸质文档管理的智能化解决方案。用户只需拍照上传，AI 自动识别文档类型、提取关键信息、智能分类，并提供保留期限建议。解决「纸质文档囤积但找不到」「不知道该保留什么」「数字化整理太麻烦」三大痛点。

## 功能设计

### 核心功能

1. **智能拍照归档**
   - 拍照自动识别文档类型（证件/合同/发票/证书/保单）
   - 自动裁剪、矫正、增强图像质量
   - 支持批量拍照，连续录入

2. **智能分类与标签**
   - 自动提取关键信息（日期、机构、金额、有效期）
   - 智能标签系统（按类型、时间、重要性）
   - 全文 OCR + 搜索，3 秒定位文件

3. **保留建议系统**
   - 基于文档类型和法律要求，智能建议保留期限
   - 到期提醒可销毁，减少囤积焦虑
   - 重要文档永久标记，永不错删

4. **安全存储**
   - 端到端加密云存储
   - 生物识别解锁（Face ID / 指纹）
   - 敏感文档本地离线模式

5. **一键导出**
   - 办事时一键导出电子版（PDF/图片）
   - 支持按场景打包（签证办理、房产交易）
   - 分享临时链接，可设置过期时间

### 辅助功能

- 文档健康检查（即将过期、重复文件提醒）
- 家庭共享（多成员协作管理）
- 纸质原件收纳建议（文件夹标签打印）

## 技术方案

### 数据层

- **OCR 服务**：百度 OCR / 腾讯云 OCR API（准确率 95%+）
- **文档分类**：CLIP + 规则引擎，初期支持 10-15 类文档
- **存储架构**：
  - 敏感文档：本地 SQLite + 加密
  - 常规文档：云端加密存储（阿里云 OSS / 腾讯云 COS）
  - 搜索引擎：
    - 本地搜索：SQLite FTS5（轻量高效，移动端原生支持）
    - 云端搜索：Meilisearch（比 Elasticsearch 更轻量，适合中小规模）

### 逻辑层

- **文档识别引擎**：
  1. 图像预处理（去噪、矫正、增强）
  2. 图片哈希去重（避免重复文档重复计费）
  3. OCR 识别（文字 + 版面分析）
  4. 分类模型（文档类型 + 关键字段提取）
  5. 用户确认机制（识别质量差时跳过计费）
  6. 保留规则匹配（法律数据库 + 自定义规则）

- **安全机制**：
  - 传输加密（HTTPS / TLS 1.3）
  - 存储加密（AES-256）
  - 访问控制（JWT + 生物识别）

### 展示层

- **移动端优先**：iOS / Android 原生应用
- **核心交互**：
  - 相机扫描界面（支持连续拍照）
  - 分类结果确认（可手动调整）
  - 搜索界面（支持语音搜索）
  - 保留期限日历视图

- **数据可视化**：
  - 文档类型分布图
  - 即将过期文档列表
  - 存储空间占用统计

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| **隐私安全** | 端到端加密 + 本地离线模式 + 安全审计 |
| **OCR 识别准确率** | 图像预处理 + 用户确认机制 + 反馈学习 |
| **低频需求** | 从高价值场景切入（签证、房产）+ 家庭共享提高活跃度 |
| **竞品压力** | 差异化定位：保留建议 + 智能标签 + 更简洁交互；深度分析扫描全能王、Adobe Scan、Microsoft Lens 的功能缺口，主打「智能管理」而非「扫描工具」 |
| **用户囤积心理** | 强调「到期提醒可销毁」+ 存储空间可视化 |
| **照片质量参差** | 拍照引导 + 图像增强算法 + 重新拍摄提示 |
