# AI 神经多样性家庭智囊 — NeuroDiverse AI

> **一句话卖点**：ADHD/自闭症孩子的家长，在情绪崩溃的深夜打开手机，10 分钟内获得基于行为分析的个性化应对策略 + 情绪安抚，从独自焦虑到科学养育

## 概述

基于 Issue #308 创建。中国约有 2300 万 ADHD 儿童、1000 万自闭症谱系障碍儿童，涉及家庭 1500-2000 万户。家长面临专业知识匮乏、个性化策略不足、情绪管理困难、资源分散、家庭关系紧张、社交孤立六大核心痛点。现有解决方案高度分散——医院排队 2-3 个月、康复机构费用 ¥500-1000/次、家长互助群信息质量参差不齐、通用育儿 APP 对神经多样性场景支持为零。

**核心价值**：不是替代专业治疗，而是成为家长身边的「AI 家庭行为分析师 + 情绪教练」，7×24 小时提供科学应对策略和情感支持。

本产品整合了 Issue 原文 5 大功能模块（行为分析引擎、干预方案生成器、情绪支持系统、资源整合平台、家庭协作网络），并吸收了评审团队（产品 9.0/10、技术 8.5/10、商业 9.2/10、增长 9.3/10）的关键建议：
- 分阶段降低技术复杂度（文本优先 → 语音 → 视觉）
- 本地优先存储 + 联邦学习保护儿童隐私
- B2B2C 机构合作降低获客成本
- 内容专业性与专家团队深度绑定

## 功能设计

### 核心功能

**1. 智能行为记录与分析**
- 文本快速记录（MVP）：家长用自然语言描述孩子行为（"今天在学校又推了同学"），AI 自动提取触发因素、行为类型、严重程度
- 语音记录（Phase 2）：支持语音描述，自动转文字并分析
- 行为模式识别：基于历史数据发现规律（"每周一早上容易情绪崩溃"）
- 触发因素图谱：时间/地点/人物/活动的关联分析

**2. 个性化干预方案生成器**
- 基于神经类型（ADHD-I/ADHD-HI/ASD Level 1-3）生成定制策略
- 效果追踪：每次策略实施后记录效果（1-5 星），持续优化
- 预设策略模板库（MVP 聚焦 3 大类）：
  - 情绪调节策略（深呼吸、感官减负、安全空间）
  - 专注力训练（番茄钟改良版、任务拆解、视觉提示卡）
  - 社交技能练习（角色扮演脚本、情景卡片、社交故事）

**3. 家长情绪支持系统**
- 情绪签到：每日快速记录家长情绪状态（emoji 选择 + 一句话）
- AI 共情回复：基于情绪状态提供个性化鼓励和具体建议
- 正念练习库：3-5 分钟音频引导（呼吸练习、身体扫描、自我关怀冥想）
- 崩溃急救包：预设的紧急应对方案（"我现在很生气，孩子也在哭" → 3 步冷静流程）

**4. 专业资源整合**
- 本地化推荐：根据地理位置推荐康复机构、治疗师、支持团体
- 专家知识库：儿童心理学、ABA 行为分析、感统训练等领域的结构化内容
- 持续教育：每周推送最新研究摘要 + 实用解读

**5. 家庭协作网络**
- 多角色看板：父母、祖父母、老师各自的任务和记录
- 统一策略视图：全家执行一致的干预方案
- 进展周报：自动生成家庭进展总结，可分享给治疗师

## 用户冷启动策略

### 新用户引导流程（Day 0）
1. **神经类型评估问卷**（5 分钟）：基于 SNAP-IV 量表简化版 + 社交反应量表（SRS-2）简化版，快速建立孩子画像
2. **痛点选择**：勾选当前最大困扰（情绪爆发 / 专注力差 / 社交困难 / 睡眠问题 / 学校冲突）
3. **首次体验**：记录今天的一个行为事件 → 即刻获得 1 条针对性建议（确保"啊，这真的有用"的瞬间）
4. **7 天新手计划**：每日 1 个小任务（记录行为 / 尝试 1 个策略 / 情绪签到），降低启动门槛

### 留存机制
- **进步可视化**：每周自动生成"本周行为趋势报告"，对比上周改善率
- **策略效果追踪**：直观展示哪些策略有效、哪些需要调整（雷达图）
- **成就系统**：连续记录 7 天 → "细心观察者"勋章；首次成功应对崩溃 → "情绪教练"勋章
- **家长社区**：匿名分享经验（可选），找到同类型孩子的家庭，降低孤独感

### 激活策略
- **智能推送**：根据孩子行为规律，在崩溃高发时段前 30 分钟推送预防策略
- **每周洞察**：AI 生成"本周发现"（如"周三下午的行为问题可能与美术课有关"）
- **专家直播**：每月 2 场在线专家答疑（免费版可观看回放，Pro 版可提问）

## 技术方案

### 数据层
- **行为记录**：多维度行为数据（触发因素、行为类型、强度、持续时间、应对策略、效果评分）
- **用户画像**：孩子神经类型、年龄、诊断信息、敏感因素、有效策略偏好
- **知识图谱**：症状 → 干预策略 → 效果证据 的结构化知识库（Neo4j）
- **内容库**：专家审核的干预策略模板（200+ 条 MVP 目标）、正念音频（30+ 条）

### 逻辑层
```
用户输入（文本/语音）
    ↓
行为分析引擎（NER 提取触发因素 + 行为分类）
    ↓
知识图谱查询（匹配神经类型 + 年龄 + 诊断的推荐策略）
    ↓
个性化排序（基于历史效果数据 + 协同过滤）
    ↓
策略生成 + 情绪状态感知（若家长情绪低，优先安抚后给策略）
    ↓
效果追踪 → 知识图谱更新（强化学习优化推荐）
```

### 隐私架构（核心设计）
- **本地优先**：行为记录默认存储在设备本地，云端仅存加密摘要
- **端到端加密**：TLS 1.3 传输，AES-256 静态加密
- **数据最小化**：不上传原始语音/视频，仅上传文本分析结果
- **合规标准**：符合《个人信息保护法》、HIPAA（如出海）
- **用户控制**：一键导出全部数据、一键删除账户

### 分阶段技术实现

**Phase 1（MVP）：文本分析模式**
- 文本行为记录 → NLP 提取 → 规则 + LLM 混合推荐
- 延迟：< 3 秒（可接受，非实时场景）
- 成本：极低（纯文本处理）

**Phase 2：语音增强**
- 语音输入 → ASR（讯飞）→ 文本分析（同 Phase 1）
- 正念音频：预录 TTS，非实时生成
- 语音情绪分析（家长端）：Wav2Vec2 微调模型

**Phase 3：多模态升级**
- 边缘端轻量模型（100MB 以内）处理本地行为识别
- 云端复杂分析（仅上传脱敏特征向量）
- 可穿戴设备接入（心率/皮电数据，可选）

### 展示层
- **移动端 App（核心）**：React Native 跨平台，iOS + Android
- **Web 端**：详细报告、家庭协作看板、打印周报
- **小程序**：微信生态快速触达，降低下载门槛
- **治疗师后台**：授权查看患者数据，辅助临床决策

### 技术栈
| 层级 | 技术选型 | 理由 |
|------|---------|------|
| 前端 | React Native + Expo | 跨平台，快速迭代 |
| 后端 | Python + FastAPI | AI 生态完善，开发效率高 |
| 主数据库 | PostgreSQL | 结构化数据，JSON 支持好 |
| 缓存 | Redis | 会话管理、排行榜 |
| 知识图谱 | Neo4j | 症状-干预关系推理 |
| 向量库 | Milvus | 语义搜索、相似用户匹配 |
| LLM | GLM-4 / Qwen | 中文能力强，成本可控 |
| ASR | 讯飞语音（国内） | 中文识别准确率 98%+ |
| 部署 | Kubernetes + 阿里云 | 国内合规，低延迟 |

## 目标用户

### 核心用户
- **新手家长**（孩子刚确诊 0-6 个月）：最焦虑、最需要指导、付费意愿最强
- **学龄期家长**（6-12 岁）：面临学校适应、社交挑战，需要持续策略支持

### 扩展用户
- **青少年家长**（12-18 岁）：青春期 + 神经多样性，复杂度最高
- **祖父母/保姆**：通过家庭协作功能间接使用
- **特教老师/治疗师**：通过专业版管理多个学生

### 用户画像
- 年龄：28-45 岁
- 学历：本科及以上（70%+）
- 收入：家庭年收入 15-50 万（中产阶级）
- 特征：高焦虑、高信息需求、高付费意愿、重视科学育儿

## 竞争壁垒

1. **专业知识图谱壁垒**：症状-干预-效果的知识图谱需要专家团队持续建设，竞品难以快速复制（预估 12-18 个月构建时间）
2. **行为数据飞轮**：用户越多 → 行为数据越多 → 推荐越准 → 效果越好 → 口碑越好 → 用户越多（正循环）
3. **机构合作网络**：与 50+ 儿童医院、100+ 康复机构的合作关系，需要 1-2 年建立
4. **家长社区网络**：活跃的家长互助社区，用户迁移成本高（关系链、历史数据）
5. **合规与伦理壁垒**：儿童数据隐私保护的专业能力，符合法规要求的技术架构
6. **专家背书效应**：知名儿童精神科医生、特教专家的推荐和合作
