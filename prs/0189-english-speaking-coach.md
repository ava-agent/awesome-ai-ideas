# AI 英语口语陪练

> **一句话卖点**：社恐英语学习者在面试前 10 分钟，用 AI 口语陪练对话练习，从开口就结巴到自信流利聊完整场

## 概述

基于 Issue #189 创建。英语口语是中国学习者的最大痛点 —— 雅思/托福考生每年 300 万+，外企职场人数数千万，口语培训市场超 500 亿元。现有方案要么太贵（外教课 200 元/小时），要么无真实对话感（APP 录音播放），要么时间难凑（语伴网站）。本产品通过「场景化对话 + 实时反馈 + 难度自适应」解决这些痛点。

**核心价值**：不是教你语法，而是陪你开口练习，给即时、专业、零尴尬的反馈。

## 功能设计

### 核心功能
1. **场景化对话** - 雅思 Part 1/职场面试/商务会议等真实场景模板
2. **实时发音反馈** - 检测发音准确度、语调、连读，即时纠正
3. **语法纠错** - 指出时态、单复数、介词等常见错误，给出地道表达
4. **难度自适应** - 根据用户水平动态调整词汇难度和语速
5. **学习档案** - 记录错误模式、进步曲线，生成个性化学习计划

### 差异化优势
| 现有方案 | 本产品 |
|---------|--------|
| 外教课 ¥200/小时，太贵 | ¥29/月无限对话 |
| Duolingo 无真实对话感 | AI 实时对话，像真人语伴 |
| Cambly 需预约，社恐压力大 | 随时练，零尴尬，AI 不评判 |
| 录音 APP 只能听，不能互动 | 实时反馈 + 纠错 + 进步追踪 |
| 语伴网站时间难凑 | 24/7 在线，想练就练 |

## 技术方案

### 数据层
- **对话记录**：多轮对话历史、错误标注（本地 + 云端备份）
- **用户画像**：水平等级、错误模式、学习目标、进步曲线
- **场景模板库**：雅思高频话题、职场场景、日常对话

### 逻辑层
```
语音输入 → ASR（Whisper/讯飞）→ LLM（GLM-4/kimi-k2.5）→ TTS（ElevenLabs）
                ↓                          ↓
         场景模板匹配              语法/发音纠错
                ↓                          ↓
         难度自适应引擎            地道表达推荐
```

**关键技术点**：
- **低延迟**：ASR+LLM+TTS 全流程 < 2s，保证对话流畅
- **发音评分**：接入 Speechace 或自建音素级评分模型
- **上下文保持**：维护场景状态，支持多轮连贯对话

### 展示层
- **移动端 App**：随时随地练习（核心场景）
- **Web 端**：详细学习报告、历史记录
- **耳机模式**：走路/通勤时语音对话

### 技术栈
- 语音识别：Whisper API / 讯飞语音
- 文本生成：GLM-4 / kimi-k2.5
- 语音合成：ElevenLabs / Azure TTS
- 前端：React Native（跨平台）
- 后端：Node.js + PostgreSQL + Redis

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| 低延迟要求高（<2s），技术挑战大 | Phase 1 先做「上传录音分析」，降低实时性要求 |
| 通用 AI 也能练口语，差异化不足 | 强调「场景模板」「发音评分」「学习档案」专业功能 |
| 用户可能「练完就跑」，留存低 | 设计进步曲线、成就系统、长期学习计划 |
| 外教/培训机构可能抵制 | 定位为「课后练习工具」，不是替代，是补充 |
| 发音评分准确率难保证 | 先用第三方 API（Speechace），积累数据后自建模型 |

## 目标用户

- **核心用户**：雅思考生、外企求职者、职场人士（跨国会议）
- **扩展用户**：留学预备党、英语爱好者、K12 学生

## 成功指标

- **MVP 验证**：100 个付费用户，NPS > 40
- **增长目标**：6 个月内 1,000 付费用户，MRR > ¥30,000
- **留存目标**：30 日留存率 > 40%
- **效果指标**：用户平均口语分数提升 0.5 分（雅思）

## 竞争壁垒

1. **场景数据积累**：不同场景下的对话模板和用户错误模式
2. **自适应算法**：越用越懂用户水平和学习曲线
3. **学习档案**：用户迁移成本（历史数据、进步曲线）
4. **情感连接**：用户和 AI 教练的「师生关系」建立
