# AI 英语口语陪练

> **一句话卖点**：社恐英语学习者在面试前 10 分钟，用 AI 口语陪练对话练习，从开口就结巴到自信流利聊完整场

## 概述

基于 Issue #189 创建。英语口语是中国学习者的最大痛点 —— 雅思/托福考生每年 300 万+，外企职场人数数千万，口语培训市场超 500 亿元。现有方案要么太贵（外教课 200 元/小时），要么无真实对话感（APP 录音播放），要么时间难凑（语伴网站）。本产品通过「场景化对话 + 实时反馈 + 难度自适应」解决这些痛点。

**核心价值**：不是教你语法，而是陪你开口练习，给即时、专业、零尴尬的反馈。

## 竞品分析

### 主要竞争对手

| 产品 | 定位 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|------|-------------|
| **Cambly** | 真人外教 1v1 | 真实对话体验 | ¥200+/小时，需预约，社恐压力大 | 随时可用，零尴尬，价格 1/10 |
| **Duolingo** | 游戏化语言学习 | 用户量大，趣味性强 | 无真实口语对话，偏书面练习 | AI 实时对话，专业发音反馈 |
| **ELSA Speak** | 发音评分 | 发音分析专业 | 单项功能，无对话场景 | 场景化对话 + 发音反馈一体化 |
| **Speak** | AI 口语陪练 | 软银投资，技术成熟 | 主要面向韩国市场，本土化不足 | 中国本土化，雅思/职场场景优先 |
| **Call Annie** | AI 视频对话 | 实时对话流畅 | 通用场景，无学习功能 | 学习导向，进步追踪，错误纠正 |

### 我们的护城河
1. **中国本土化**：
   - 雅思口语 Part 1-3 高频话题 50+ 个（Hometown/Study/Work/Travel 等）
   - 外企面试真题库 200+ 题（Amazon/Microsoft/Google 中国面试题）
   - 中国学生常见错误模式（th/s 混淆、时态错误、中式表达）
   - 符合中国用户习惯的 UI（微信登录、支付宝支付）

2. **场景垂直**：不只练口语，而是「为特定场景做准备」

3. **学习闭环**：对话 → 反馈 → 学习计划 → 进步追踪 → 下次练习

## 功能设计

### 核心功能
1. **场景化对话** - 雅思 Part 1/职场面试/商务会议等真实场景模板
2. **实时发音反馈** - 检测发音准确度、语调、连读，即时纠正
3. **语法纠错** - 指出时态、单复数、介词等常见错误，给出地道表达
4. **难度自适应** - 根据用户水平动态调整词汇难度和语速
5. **学习档案** - 记录错误模式、进步曲线，生成个性化学习计划

### 差异化优势
| 现有方案 | 本产品 |
|---------|--------|
| 外教课 ¥200/小时，太贵 | ¥59/月无限对话 |
| Duolingo 无真实对话感 | AI 实时对话，像真人语伴 |
| Cambly 需预约，社恐压力大 | 随时练，零尴尬，AI 不评判 |
| ELSA Speak 只有发音评分 | 对话 + 发音 + 语法 + 词汇一体化 |
| Speak 韩国市场为主 | 中国本土化，雅思/职场优先 |
| 录音 APP 只能听，不能互动 | 实时反馈 + 纠错 + 进步追踪 |
| 语伴网站时间难凑 | 24/7 在线，想练就练 |

## 用户冷启动策略

### 新用户引导流程
1. **水平测试**（2 分钟）：AI 简单对话，评估当前水平
2. **目标选择**：雅思/面试/日常会话/商务英语
3. **首次体验**：从最简单场景开始（自我介绍），保证成功感
4. **详细反馈**：前 3 次对话提供超详细反馈，建立信心

### 留存机制
- **进步可视化**：每次练习后展示进步曲线、错误减少比例
- **成就系统**：连续打卡奖励、场景通关勋章、发音满分成就
- **挑战任务**：每周挑战（如「完成 3 次面试场景」），解锁新内容
- **社交元素**：学习小组、口语挑战赛（**完全可选**，默认关闭，保护社恐用户；用户可在设置中主动开启）

### 激活策略
- **智能提醒**：根据用户习惯，在最佳时间推送练习提醒
- **场景推荐**：结合用户目标，推荐今日最适合练习的场景
- **复习机制**：间隔重复，针对错误点推送复习任务

## 技术方案

### 数据层
- **对话记录**：多轮对话历史、错误标注（本地 + 云端备份）
- **用户画像**：水平等级、错误模式、学习目标、进步曲线
- **场景模板库**：雅思高频话题、职场场景、日常对话

### 逻辑层
```
语音输入 → ASR（Whisper/讯飞）→ LLM（GLM-4/kimi-k2.5）→ TTS（ElevenLabs）
                ↓                          ↓
         场景模板匹配              语法/发音纠错
                ↓                          ↓
         难度自适应引擎            地道表达推荐
```

### 低延迟技术方案（分阶段实现）

**Phase 1：录音分析模式**（降低技术门槛，快速验证）
- 用户录音 → 上传 → 服务器处理 → 返回反馈
- 延迟：5-10 秒
- 优点：技术简单，成本低，快速上线
- 目标：验证需求，收集数据

**Phase 2：实时对话模式**（核心体验）
- 流式 ASR：边说边识别，延迟 < 500ms
- 流式 LLM：Token-by-token 生成
- 流式 TTS：音频流式返回
- 目标延迟：< 2 秒端到端

**延迟分配目标**：
| 环节 | 目标延迟 | 优化策略 |
|------|---------|---------|
| ASR 识别 | < 500ms | 讯飞实时语音识别（国内优化） |
| LLM 生成 | < 800ms | 流式输出 + 常见回复缓存 |
| TTS 合成 | < 500ms | ElevenLabs Turbo / 讯飞在线 |
| 网络传输 | < 200ms | 就近服务器部署 |
| **总计** | **< 2,000ms** | 并行处理 + 流水线优化 |

**技术选型**：
- 流式 ASR：讯飞实时语音识别（国内）/ Deepgram（海外）
- 流式 LLM：GLM-4 / Qwen-Max 流式输出
- 流式 TTS（国内）：讯飞在线语音合成 / Azure 中国 TTS / 云知声
- 流式 TTS（海外）：ElevenLabs Turbo v2.5（延迟 < 300ms）

**国内网络环境适配**：
- 主选：讯飞语音全家桶（ASR + TTS），国内延迟稳定 < 800ms
- 备选：Azure 中国区（需备案），延迟约 500-1000ms
- 降级方案：国内用户优先用讯飞，海外用户用 ElevenLabs

**成本控制**：
- 免费版限制时长，避免滥用
- Pro 版提供发音评分（成本高）
- 智能缓存：常见回复预生成

### 展示层
- **移动端 App**：随时随地练习（核心场景）
- **Web 端**：详细学习报告、历史记录
- **耳机模式**：走路/通勤时语音对话

### 技术栈
- 语音识别：Whisper API / 讯飞语音
- 文本生成：GLM-4 / kimi-k2.5
- 语音合成：ElevenLabs / Azure TTS

## 资源需求

- 语音识别：Whisper API（$0.006/分钟）

- 语音合成：ElevenLabs（$5/月起）
- 发音评分：Speechace API（$0.02/次）

### 成本分析（单用户/月）
假设活跃用户每天练习 15 分钟（调整后更保守）：
- 讯飞 ASR: ¥0.033/分钟 × 15 × 30 = ¥15
- GLM-4: ¥0.01 × 8K tokens × 30 = ¥240（可优化，实际用量更低）
- 讯飞 TTS: ¥0.002/次 × 30 × 30 = ¥1.8
- Speechace（Pro 版）: $0.02 × 5 × 30 = $3（¥22）

**成本假设依据**：
- **8K tokens/天来源**：单次对话平均 10 轮，每轮用户输入约 100 tokens + AI 回复约 300 tokens = 400 tokens，每天 20 次对话 = 8,000 tokens
- **15 分钟/天**：基于竞品用户行为数据，口语练习类产品日均在 10-20 分钟
- **Speechace 5 次/天**：Pro 用户每 3 次对话请求 1 次发音评分

**总成本估算**：
- 免费用户：¥20-30/月（限制发音评分）
- Pro 活跃用户：¥40-50/月

**优化策略**：
- 免费版：每日 5 分钟，无发音评分
- Pro 版：¥79/月，利润空间 35-50%
- 长期：积累数据后自建发音评分模型

### 人力
- 全栈开发 × 1（8-10 周）
- UI/UX 设计 × 1（4 周）
- 内容运营 × 1（场景模板制作）

### 预算估算
- MVP 开发：¥80,000（调整后更现实）
- 服务器 + API：¥5,000/月
- 推广预算：¥20,000（冷启动）

### 获客成本分析（CAC/LTV）
**获客成本（CAC）**：
- 内容营销（小红书/知乎）：¥30-50/用户
- 付费广告（抖音/微信）：¥80-150/用户
- 口碑推荐：¥10-20/用户（激励成本）
- **平均 CAC 目标**：¥50-80/用户

**用户生命周期价值（LTV）**：
- Pro 用户平均付费周期：4 个月
- LTV = ¥79 × 4 = ¥316
- LTV/CAC 比例：4-6 倍（健康）

**回收周期**：
- CAC ¥80，月利润 ¥40（¥79 - ¥40 成本）
- 回收周期：2 个月

**优化策略**：
- 优先内容营销，降低 CAC
- 年付方案提高 LTV
- 提高留存率，延长付费周期

| 风险 | 缓解措施 |
