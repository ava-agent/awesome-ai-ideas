# AI 订阅管家 - 智能追踪你的数字支出

> 自动分析订阅生态，发现闲置服务，每月省下冤枉钱

## 功能设计

### 核心功能
- **支出透视** - 自动扫描银行账单/邮件，汇总所有订阅费用，生成月度支出报告
- **使用分析** - 检测订阅服务的实际使用频率，标记"僵尸订阅"
- **重复检测** - 智能识别功能重叠的订阅（如多个音乐App、多个云存储），推荐合并方案
- **智能建议** - 基于使用数据，推荐取消/降级方案，量化每月可省金额

### 辅助功能
- 订阅日历 - 提醒即将续费的服务
- 价格追踪 - 监控订阅涨价通知
- 家庭版 - 统一管理家庭成员订阅

## 技术方案

### 数据层
- **邮件解析**：IMAP 扫描订阅确认邮件（Netflix、Spotify、Adobe 等）
- **账单识别**：PDF/截图 OCR + 规则引擎，提取订阅名称、金额、周期
- **本地存储**：SQLite + 加密，敏感数据不上云

### 逻辑层
- **订阅识别**：NER 模型 + 规则库，识别账单中的订阅项
- **使用频率**：API 对接（如 RescueTime）或手动标记
- **智能建议**：基于规则的推荐引擎 + LLM 生成个性化建议

### 展示层
- **Web Dashboard**：React + Tailwind，图表展示支出趋势
- **移动端**：React Native，重点做提醒和快速操作
- **隐私优先**：核心计算在本地完成，可选云端同步

## 实现步骤

### Phase 1: MVP (4 周)
1. 邮件扫描模块 - 支持主流订阅服务确认邮件解析
2. 手动导入账单 - PDF/截图识别，提取订阅信息
3. 基础 Dashboard - 订阅列表、月度支出汇总
4. 取消提醒 - 续费前 3 天推送通知

### Phase 2: 智能化 (3 周)
1. 使用频率追踪 - 对接 RescueTime/Screen Time API
2. 重复检测算法 - 基于服务分类和功能标签
3. 智能建议生成 - LLM 生成个性化省钱方案

### Phase 3: 增值功能 (2 周)
1. 订阅日历视图
2. 价格变动追踪
3. 家庭版基础功能

## 资源需求

### API & 服务
- **OCR 服务**：Tesseract (本地) 或 Google Vision API
- **LLM**：Claude/GPT-4 用于建议生成（可选，可用本地模型替代）
- **邮件服务**：IMAP 协议（无需第三方服务）

### 开发资源
- **前端**：1 人 * 4 周
- **后端**：1 人 * 4 周
- **设计**：0.5 人 * 2 周

### 运营成本
- 云端存储：$10-20/月（可选，支持纯本地模式）
- LLM API：$50-100/月（如使用云端 LLM）

## 风险与缓解

| 风险 | 缓解措施 |
|------|---------|
| 银行数据获取困难 | 先做邮件扫描 + 手动导入，验证需求 |
| 隐私担忧 | 强调本地处理，开源代码可审计 |
| 订阅识别误判 | 用户可手动校正，反馈优化模型 |

## 预期价值

- **用户价值**：平均每月节省 20-30% 订阅支出
- **市场验证**：竞品 Truebill 已被 Rocket Money 收购，证明市场需求
- **差异化**：隐私优先 + 本地处理，适合注重隐私的用户
