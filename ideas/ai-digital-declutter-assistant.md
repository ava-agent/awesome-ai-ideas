# AI 数字断舍离助手

> 智能分析数字生活，一键清理数字囤积

## 功能设计

### 核心功能
- **存储分析**：识别大文件、重复照片、过期下载
- **通知整理**：分析 App 推送习惯，建议关闭低价值通知
- **订阅清理**：检测长期未读的订阅/频道，建议退订
- **文件归类**：自动整理 Downloads/Desktop 混乱文件
- **断舍离报告**：每周生成清理建议，一键执行

### 差异化亮点
- 基于用户行为的智能决策系统（非简单清理）
- 「决策辅助」模式：推荐是否保留，用户最终确认
- 断舍离报告天然适合社交分享

## 技术方案

### 数据层
- **文件扫描**：Electron (Node.js fs) / Android Storage Access Framework
- **使用统计**：系统日志分析（Android UsageStatsManager）
- **订阅检测**：邮件解析 + 账单识别（Regex + LLM）

### 逻辑层
- **重复检测**：pHash/dHash 图片指纹 + SimHash 文本指纹
- **价值评估**：基于访问频率、文件大小、类型权重
- **决策引擎**：规则系统 + LLM 辅助判断

### 展示层
- **桌面端**：Electron（跨平台）
- **移动端**：先推 Android（系统权限更开放）
- **报告生成**：Markdown 模板 + 图表（Chart.js）

## 实现步骤

### MVP (v0.1) - 存储分析
1. 实现文件扫描引擎（Electron）
2. 重复文件检测（图片 + 文档）
3. 大文件识别（>100MB）
4. 基础清理建议 UI

### V1.0 - 智能决策
1. 添加「使用频率」维度分析
2. 通知分析（Android）
3. 订阅检测（邮件解析）
4. 周报生成 + 一键清理

### V2.0 - 生态扩展
1. iOS 支持（受限功能）
2. 云存储整合（iCloud/Google Drive）
3. 社交分享功能
