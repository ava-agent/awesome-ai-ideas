# AI 智能合同阅读助手

> 让普通人也能看懂法律条款，拍照即分析，风险一目了然

## 功能设计

### 核心功能
- **条款解读** - 将法律术语翻译成通俗易懂的人话
- **风险标注** - 自动高亮潜在陷阱和不公平条款
- **要点总结** - 关键信息一目了然（价格、期限、解约条件、违约责任）
- **对比分析** - 多个选项横向对比（如两份租房合同）
- **修改建议** - 提供可协商的条款修改建议

### 支持场景
- 租房合同（MVP 优先）
- 保险条款
- 服务协议
- 劳动合同
- 购房合同

## 技术方案

### 数据层
- **文档解析**：OCR（PaddleOCR/Tesseract）+ PDF 解析（pdfplumber）
- **条款知识库**：
  - 租房合同常见条款库
  - 保险条款解读库
  - 消费者权益保护法相关条文
- **存储**：SQLite 本地存储（隐私优先）

### 逻辑层
- **条款识别**：LLM + 规则引擎混合
  - 规则引擎处理标准化条款（价格、期限、面积）
  - LLM 处理非标条款解读
- **风险评估**：
  - 风险知识图谱（条款类型 → 风险等级 → 典型陷阱）
  - 不公平条款检测模型
- **建议生成**：RAG + 法律知识库

### 展示层
- **移动端优先**：React Native / Flutter
- **可视化标注**：
  - 🔴 高风险条款（红色高亮）
  - 🟡 注意条款（黄色标记）
  - 🟢 正常条款（绿色通过）
- **报告导出**：PDF 分析报告

## 实现步骤

### Phase 1: MVP (2-3周)
1. 租房合同场景专属
2. 拍照/上传 PDF 解析
3. 基础条款识别（租金、押金、期限、违约责任）
4. 风险等级标注
5. 简化版要点总结

### Phase 2: 多场景扩展 (4周)
1. 保险条款支持
2. 劳动合同支持
3. 对比分析功能
4. 修改建议生成

### Phase 3: 商业化 (持续)
1. B2B 中介批量审核
2. 律师服务引流
3. API 开放

### 免责声明
> 本工具提供的分析仅供参考，不构成法律建议。如有法律纠纷，请咨询专业律师。

## 商业模式

### B2C
- **Freemium**：3 份免费分析/月
- **订阅制**：¥29.9/月 无限分析

### B2B
- 房产中介批量合同审核：¥999/月
- 保险公司条款合规检测
- 企业 HR 劳动合同审核

### 增值服务
- 一键生成律师咨询函
- 连接律师服务平台（抽佣）

## 竞品分析

| 竞品 | 优势 | 劣势 |
|------|------|------|
| 法大大 | 电子签名 + 合规 | 无 AI 解读 |
| 法天使 | 合同库丰富 | 无智能分析 |
| LegalZoom | 海外成熟 | 无中文支持 |

**差异化**：「普通人视角」+「中文法律环境」是蓝海

## 风险与应对

| 风险 | 应对策略 |
|------|----------|
| 法律责任边界 | 明确「仅供参考」+ 引流律师服务分摊风险 |
| 准确性要求高 | 高风险条款人工复核 + 用户反馈机制 |
| B2C 付费意愿弱 | 先做 B2B（中介、平台）验证商业模式 |

## 成功指标

- **准确率**：条款识别准确率 > 95%
- **风险发现率**：不公平条款召回率 > 90%
- **用户满意度**：NPS > 50
- **商业化**：B2B 付费转化率 > 5%
