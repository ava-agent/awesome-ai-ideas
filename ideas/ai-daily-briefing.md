# AI 每日简报助手

> 每天早上，AI 为你生成个性化的技术新闻摘要

## 核心功能

1. **多源聚合** - RSS、HN、GitHub Trending、Twitter
2. **智能筛选** - 根据兴趣和阅读历史过滤
3. **摘要生成** - 提取关键信息，去除噪音
4. **多格式输出** - 文字、音频、邮件

## 使用场景

- 🌅 晨间 5 分钟快速了解行业动态
- 🚗 通勤时收听播客版
- 📧 邮件推送到收件箱

## 技术栈

- 数据采集：RSS + API
- 内容处理：LLM (GLM-5/Claude)
- 语音合成：ElevenLabs / Azure TTS
- 推送：Telegram / Email / Podcast
