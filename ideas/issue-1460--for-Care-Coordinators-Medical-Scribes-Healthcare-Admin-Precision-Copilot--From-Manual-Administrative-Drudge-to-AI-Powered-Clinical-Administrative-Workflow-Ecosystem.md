# 💡 [for Care Coordinators & Medical Scribes] Healthcare Admin Precision Copilot - From Manual Administrative Drudge to AI-Powered Clinical Administrative Workflow Ecosystem

## Issue #1460

---

## 🎯 问题
当前临床 AI 更擅长“写得像”，但不一定“交得出去”；医生和行政团队还要手工补齐 payer 要求、编码、回访安排和模板字段；大厂方案多面向大型医院系统，专科诊所和轻量团队缺少更小、更窄、更快落地的产品。

## 🚀 功能
构建 Healthcare Admin Precision Copilot，面向诊后行政流程的精度层，把就诊转录、EHR 摘要、payer 规则拼成可提交的 prior auth / referral / appeal / follow-up 包，内置字段级检查器、payer checklist、缺项提示、提交前 QA。

## 🛠️ 方案
1. 面向诊后行政流程的精度层：把就诊转录、EHR 摘要、payer 规则拼成可提交的 prior auth / referral / appeal / follow-up 包
2. 内置字段级检查器、payer checklist、缺项提示、提交前 QA
3. 可先从风湿/肿瘤/精神科这类 prior auth 重的专科切入
4. 验证指标：人工补字段时间、首次提交通过率、平均周转天数、医生二次编辑比例
