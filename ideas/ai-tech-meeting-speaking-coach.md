# AI 技术会议发言教练

> **一句话卖点**：技术会议沉默者在架构评审会上，用 AI 发言教练收到实时提示，从全程闭麦到抓住时机提出关键问题

## 概述

基于 Issue #125 创建的详细设计文档。

技术人员经常在会议中有想法却不敢说——担心说错显得不专业、不知道如何组织语言、怕打断别人节奏、等想好了话题已经过去。AI 技术会议发言教练帮助技术人员从「全程闭麦」到「主动发言」，通过会议前准备、实时提示、发言演练、渐进式成长四步实现蜕变。

## 功能设计

### 核心功能

1. **会前准备模块**
   - 上传会议议题材料（PPT、文档、代码）
   - AI 生成 3-5 个关键提问角度
   - 提供专业表达模板（STAR、PREP 等结构）
   - 推送今日会议话题卡片

2. **实时提示模块**
   - 会议实时转录（Whisper API / 讯飞语音）
   - 话题追踪与时机识别（停顿、提问、征求意见信号）
   - 推送可发言窗口 + 参考话术
   - 个性化建议（基于用户技术专长）

3. **会后复盘模块**
   - 发言质量分析（结构、清晰度、影响力）
   - 对比「AI 建议话术」vs「实际发言」
   - 未发言想法的模拟对话练习
   - 进步曲线追踪

4. **渐进式成长系统**
   - Level 1：附议型发言（补充他人观点）
   - Level 2：补充型发言（提供额外信息）
   - Level 3：主导型发言（提出新观点）
   - 根据用户水平自动调整建议难度

## 技术方案

### 数据层

- **会议数据**：实时语音流 → Whisper API 转写 → 结构化存储
- **用户画像**：技术栈、擅长领域、发言风格偏好
- **话题库**：技术领域常见讨论话题 + 关键词
- **话术模板**：不同场景的表达模板（技术评审、架构讨论、代码评审）

### 逻辑层

- **语音识别引擎**：Whisper API（英文）+ 讯飞语音（中文）
- **话题追踪**：LLM 实时分析会议主题 + 关键词提取
- **时机识别**：检测「停顿」「提问」「征求意见」等信号
- **话术生成**：基于会议内容 + 用户背景 + 技术领域生成个性化建议
- **进步追踪**：多维度评分（结构、清晰度、技术准确性、影响力）

### 展示层

- **桌面客户端**：Electron / Tauri（跨平台）
- **浏览器扩展**：支持 Web 会议（Zoom、腾讯会议 Web 版）
- **移动端**：会议前推送话题卡片，会后查看复盘报告

## 资源需求

### API 服务

- **语音识别**：OpenAI Whisper API / 讯飞语音 API
- **LLM**：GPT-4 / Claude / kimi-k2.5（中文场景）
- **向量数据库**：Pinecone / Milvus（话题库检索）

### 开发资源

- 前端：2 人（桌面客户端 + 浏览器扩展）
- 后端：1 人（API 服务 + 数据处理）
- AI：1 人（模型调优 + 提示工程）

### 第三方集成

- 会议平台：Zoom / 腾讯会议 / 钉钉（可选）
- 日历：Google Calendar / Outlook（会议同步）

## 变现模式

### Freemium 模式

- **免费版**：每周 1 次会前准备 + 会后复盘
- **专业版**：¥29.9/月，无限次数 + 实时提示
- **团队版**：¥199/人/年，团队会议文化培训 + 管理者仪表盘

### B2B2C 模式

- 与技术社区（掘金、V2EX、CSDN）合作，作为会员权益
- 与企业培训平台合作，作为线上练习工具

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| **实时性挑战**：语音识别 + LLM 处理延迟可能错过发言窗口 | 提前准备话题卡片，不完全依赖实时生成；优化模型响应速度 |
| **会议隐私**：会议内容可能涉及商业机密 | 提供「本地处理」选项，敏感内容不上云；明确数据保留政策 |
| **企业 IM 集成门槛高**：飞书/钉钉 API 限制 | 先做 Web 会议（Zoom、腾讯会议 Web 版）和本地录制上传 |
| **用户依赖性**：过度依赖 AI 提示 | 设计「渐进式训练」，逐步减少 AI 提示频率；提供离线练习模式 |
| **技术准确性**：话题识别可能出错 | 允许用户手动标注感兴趣话题；持续学习用户偏好 |

## 关键指标

- **北极星指标**：用户发言次数变化（会议前 vs 会议后）
- **参与度指标**：每周使用会议数、会前准备完成率
- **效果指标**：用户自评发言信心提升、同事反馈改善率
- **商业指标**：付费转化率、月留存率
