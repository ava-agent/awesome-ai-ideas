# 💡 AI 语言智慧陪练 - 从哑巴外语不敢开口到自信流利对话

## 📋 项目概述

AI语言智慧陪练是一个基于大语言模型的智能语言学习助手，专注于解决语言学习者"哑巴外语"的核心痛点，帮助学习者从不敢开口到自信流利对话。

## 🎯 核心痛点

### 现状问题
- **不敢开口**：怕说错、怕被嘲笑，宁愿保持沉默
- **缺乏练习机会**：找不到合适的对话伙伴
- **反馈不及时**：无法及时纠正发音和语法错误
- **学习效率低**：重复练习相同内容，缺乏个性化指导

### 用户画像
- **职场人士**：需要提升商务外语能力，但时间有限
- **学生党**：考试需要口语能力，但练习机会少
- **出国人群**：需要快速适应外语环境
- **兴趣学习者**：想学外语但缺乏系统练习

## 🚀 核心功能

### 1. 智能对话系统
- **多场景对话**：商务、旅游、日常、学术等场景
- **难度自适应**：根据用户水平调整对话难度
- **话题多样性**：覆盖1000+日常话题
- **角色扮演**：模拟真实对话场景（面试、点餐、问路等）

### 2. 实时反馈系统
- **发音评估**：基于AI的发音准确度评分
- **语法纠正**：实时指出语法错误并给出正确表达
- **词汇建议**：推荐更地道的表达方式
- **流畅度分析**：评估语速、停顿、自然度

### 3. 个性化学习路径
- **水平测试**：AI评估当前语言水平
- **定制计划**：根据目标和时间制定学习计划
- **进度跟踪**：可视化学习进度和成长曲线
- **薄弱环节**：识别并强化薄弱环节

### 4. 智能复习系统
- **间隔重复**：基于艾宾浩斯遗忘曲线的智能复习
- **情景记忆**：在具体场景中复习词汇和表达
- **错误追踪**：重点复习常犯错误
- **成就感激励**：通过小目标保持学习动力

## 🛠️ 技术实现

### 架构设计
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   前端界面      │    │   AI服务层       │    │   数据存储层     │
│                │    │                │    │                │
│ - 对话界面      │◄──►│ - 对话引擎      │◄──►│ - 用户数据库    │
│ - 评估显示      │    │ - 评估模块      │    │ - 对话历史      │
│ - 进度图表      │    │ - 个性化推荐    │    │ - 词汇库        │
│ - 设置面板      │    │ - 复习算法      │    │ - 模型权重      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 核心技术栈
- **前端**：React + TypeScript + Vite
- **后端**：Python FastAPI + Pydantic
- **AI模型**：GLM-4 + Whisper (语音识别)
- **数据库**：PostgreSQL + Redis
- **部署**：Docker + Kubernetes

### 关键算法
1. **对话生成算法**：基于用户水平的历史对话生成
2. **评估算法**：语音识别 + 语法分析 + 流畅度评估
3. **推荐算法**：基于用户行为和能力的个性化推荐
4. **复习算法**：间隔重复 + 情景关联记忆
