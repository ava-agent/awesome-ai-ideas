# AI 护肤顾问 - 拍脸5秒懂肌肤，告别盲目买买买

> **一句话卖点**：护肤小白在梳妆台前，用 AI 拍照分析肤质和桌上闲置产品，从「盲目买买买」到「用对不买贵」，一年省下 ¥2000 冤枉钱

## 概述

基于 Issue #117 创建的详细设计文档。

目标用户：护肤迷茫族（不知道自己肤质、不知道产品适不适合、梳妆台堆满闲置）

核心价值：用 AI 帮用户「先懂自己再买产品」，减少护肤品浪费，提升护肤效果

## 功能设计

### 核心功能

**1. 拍脸测肤（5秒出结果）**
- 多模态模型分析肤质（干/油/混合/敏感）
- 识别皮肤问题（暗沉、毛孔、细纹、痘痘）
- 季节因素自动调整（春夏换季提醒）

**2. 现有产品盘点（差异化核心）**
- 扫产品条码识别品牌和成分
- AI 分析「这瓶适不适合你的肤质」
- 标记「应该用」「可以送人」「赶紧扔」

**3. 成分避雷雷达**
- 建立个人「成分黑名单」（如酒精过敏、视黄醇不耐受）
- 扫码自动预警「⚠️ 含有你过敏的烟酰胺」
- 生成「我的成分安全清单」

**4. 个性化护肤方案**
- 根据肤质、预算、季节生成早晚护肤流程
- 推荐搭配组合（避免成分冲突）
- 效果追踪（定期拍照对比）

### 增值功能

- **梳妆台改造挑战**：生成「我省了 ¥500」分享卡片
- **空瓶记录**：追踪产品使用周期，智能提醒回购
- **肤质档案**：记录肤质变化，生成「我的肌肤年轮」

## 技术方案

### 数据层

```
用户数据：
- 肤质档案（肤质类型、敏感成分、历史记录）
- 梳妆台清单（已有产品、使用状态、效果评价）
- 护肤记录（每日护肤打卡、肤质变化照片）

知识库：
- 成分数据库（5000+ 常见成分及其功效/风险）
- 产品数据库（对接美丽修行 API / 成分党 APP）
- 肤质-成分适配规则库（如视黄醇 vs 敏感肌）
```

### 逻辑层

```
核心模块：
1. Skin Analyzer（肤质分析）
   - GPT-4V / GLM-4V 多模态分析
   - 肤质分类 + 问题识别
   - 置信度评分

2. Product Scanner（产品识别）
   - 条码扫描 → 数据库匹配
   - 成分解析 → 成分适配分析
   - 生成「适配报告」

3. Ingredient Checker（成分检查）
   - 成分黑名单匹配
   - 成分冲突检测（如酸类 + A醇）
   - 风险等级评估

4. Routine Generator（流程生成）
   - 基于肤质 + 季节 + 预算
   - 产品搭配优化
   - 早晚流程差异化
```

### 展示层

```
前端架构：
- 微信小程序（冷启动首选）
- React Native（跨平台）
- 拍照界面优化（光线引导、对焦辅助）
- 梳妆台可视化（产品墙展示）
```
