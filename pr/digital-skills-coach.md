# AI 数字技能陪练 - 60岁也能独立操作，不再求助儿女

> **一句话卖点**：68岁的张阿姨想给孙子发红包，对着手机说「我要发微信红包」，AI 一步步语音引导她找到那个「+」号，10分钟后红包成功发出，第一次觉得智能手机没那么难

## 概述

基于 Issue #130 创建的详细设计文档。

**目标用户**：60+ 数字学习者（子女教了忘、不敢再问、逐渐脱离数字生活）

**核心价值**：AI 无限耐心的语音陪练，让老年人从「怕被嫌弃」到「独立操作不求助」

**市场背景**：
- 中国 60+ 人口 2.8 亿，智能手机普及率持续上升
- 子女愿意付费减轻教学负担
- 现有方案无法解决「遗忘」问题

## 功能设计

### 核心功能

**1. 语音引导 + 截图识别**
- 老年人语音提问：「我想发微信红包」
- AI 识别当前屏幕，告诉用户「现在点哪里」
- 语速慢、音量大、用词简单

**2. 慢节奏教学**
- 每一步等待至少 30 秒，不催促
- 长时间无操作时主动询问：「是不是卡在哪一步了？」
- 提供「休息模式」：累了可以暂停，明天继续

**3. 记忆曲线提醒**
- 根据遗忘曲线，在关键时刻主动询问「上次学的还记得吗？」
- 常用操作定期复习提醒
- 学习进度可视化（「你已经学会 5 个技能啦！」）

**4. 家庭助手模式**
- 子女预设常用任务模板（发红包、视频通话、刷抖音）
- 子女可录制操作示范视频，AI 自动提取关键步骤
- 子女查看父母的「学习进度报告」

**5. 错误恢复**
- 操作错了没关系，AI 帮忙回到正确步骤
- 「没关系，我们慢慢来」的情感支持
- 永远不会说「你怎么又忘了」

### 增值功能

- **成就系统**：完成某项任务 → 生成「我学会用微信了」卡片 → 分享到朋友圈
- **常用任务快捷入口**：首页显示「最近学的」和「子女推荐的」
- **语音备忘录**：老年人可以录下「这个怎么操作来着」，AI 回答

## 技术方案

### 数据层

```
用户数据：
- 学习档案（已学技能、学习时间、复习记录）
- 常用任务模板（子女预设 + 用户自定义）
- 操作历史（用于记忆曲线计算）

知识库：
- 常见操作引导模板库（微信、支付宝、抖音等）
- 老年人常见困惑 FAQ
- 语音识别方言适配库
```

### 逻辑层

```
核心模块：
1. Screen Analyzer（屏幕分析）
   - GPT-4V / Claude Vision 分析当前屏幕
   - 识别「可点击元素」+「当前操作位置」
   - 技术挑战：各种主题、字体大小适配

2. Voice Processor（语音处理）
   - Whisper 或讯飞语音（中文支持更好）
   - 方言和口音适配
   - 语速检测和调整

3. Guidance Engine（引导引擎）
   - 基于当前位置 + 目标任务，生成下一步指引
   - 语言简单化处理（避免技术术语）
   - 确认机制：每一步都问「完成了吗？」

4. Memory Curve Scheduler（记忆曲线调度）
   - 艾宾浩斯遗忘曲线算法
   - 在 1 天、3 天、7 天、30 天节点提醒复习
   - 根据用户实际记忆情况动态调整
```

### 展示层

```
前端架构：
- 微信小程序（老年人已熟悉微信生态）
- 或独立 App（与手机厂商预装合作）
- 超大字体、高对比度界面
- 语音优先交互（减少文字阅读）

无障碍设计：
- 支持「重复一遍」+「文字确认」双重机制
- 紧急情况一键呼叫子女
- 离线模式（基础引导功能可用）
```

### 用户访谈发现（基于 Issue #130 评论）

1. **情感支持是核心**：老年人怕的不是「学不会」，而是「被嫌弃」
2. **「慢」是关键**：每一步等待至少 30 秒，不能催促
3. **子女愿意付费**：减轻教学负担，还能看到父母学习进度
4. **分享意愿强**：「爸妈学会了」是天然社交货币
