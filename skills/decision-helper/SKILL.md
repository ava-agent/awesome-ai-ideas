# decision-helper - 决策助手 🎲

> 帮你做选择，拯救选择困难症！

## 用途

当主人不知道该选什么的时候，帮TA做决定：
- "今天吃什么？"
- "看什么电影？"
- "周末去哪玩？"
- "选 A 还是 B？"

## 使用方法

直接告诉我需要做的选择，我会：

1. **随机选择模式** - 从多个选项中随机选一个
2. **利弊分析模式** - 分析各选项的优缺点
3. **掷骰子模式** - 数字决策（1-6）

## 示例

### 随机选择

```
主人: 帮我选个吃的，火锅、烧烤、日料
咕噜: 🎲 命运选择了... 烧烤！今晚撸串去！
```

### 利弊分析

```
主人: 该不该买这个机械键盘？
咕噜: 让我帮你分析一下：

✅ 买的好处：
- 打字手感好，效率提升
- 耐用，可以用很多年
- 声音爽，ASMR 级体验

❌ 买的坏处：
- 钱包疼
- 可能吵到别人
- 又要多一个键盘

我的建议：如果你每天码字超过4小时，值得！
```

### 掷骰子

```
主人: 掷个骰子
咕噜: 🎲 掷出了 4 点！
```

## 功能清单

| 功能 | 触发词 | 说明 |
|------|--------|------|
| 随机选择 | 帮我选、选一个、吃什么 | 从多个选项中随机选 |
| 利弊分析 | 该不该、要不要、分析一下 | 分析选项优缺点 |
| 掷骰子 | 掷骰子、摇骰子、roll | 1-6 随机数 |
| 抛硬币 | 抛硬币、正面反面 | 50/50 概率 |
| 算命模式 | 算一卦、求签 | 给出神秘建议 |

## 技术实现

- 纯文本交互，无需外部依赖
- 支持中文自然语言解析
- 可扩展的决策逻辑

## 安装

无需额外安装，直接使用即可。
