# AI职场软技能导师 架构设计

## 架构概览

采用前后端分离架构：
- **后端**：Node.js + Express + TypeScript + Prisma + PostgreSQL，提供 RESTful API
- **前端**：React 18 + Vite + Material-UI 组件库，响应式设计适配桌面和移动设备
- **AI层**：基于 OpenAI GPT-4 提供场景生成、反馈评估和建议优化

## 核心模块

1. **场景模拟引擎**：基于职场常见场景（汇报、面试、冲突处理）生成对话，支持多轮交互和实时反馈；
2. **邮件优化服务**：接收用户初稿，AI生成多版本优化建议，提供语气调整和重点突出；
3. **会议准备助手**：根据会议主题和参会人信息，生成议程框架、关键问题清单和发言要点；
4. **用户进度追踪**：记录练习历史，生成能力雷达图，提供渐进式学习路径推荐。

## 数据设计

核心数据表：`users → scenarios → practice_sessions → feedback_records → progress_tracking。
使用 JWT 认证，隐私友好，用户数据端到端加密保存。

## 实施路线

MVP 阶段优先实现场景模拟和邮件优化两个核心功能，4-6 个月交付可测试版本，随后迭代企业培训功能。
