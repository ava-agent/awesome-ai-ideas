# AI职场软技能导师 - 架构方案

## 项目概览

AI职场软技能导师帮助职场新人提升沟通、邮件写作、会议准备等软技能，通过场景化AI模拟训练提供实时反馈。总评分 8.0/10，是明确的高价值刚需项目。

## 架构设计

### 技术栈

- **后端**: Node.js + Express + TypeScript + Prisma + PostgreSQL
- **AI层**: OpenAI GPT-4o + 函数调用 + RAG 知识库检索
- **前端**: React 18 + TypeScript + Vite + Tailwind CSS
- **部署**: Docker + Nginx + GitHub Actions CI/CD

### 核心分层

1. **用户层**: 注册登录/JWT认证/进度追踪/历史记录管理
2. **场景引擎**: 场景库管理（邮件/会议/面试/沟通）→ 动态生成训练内容
3. **AI交互服务**: 用户输入分析 → 实时反馈生成 → 改进建议输出
4. **知识库**: 优质范例库+常见错误库，RAG实时检索提供精准参考

### 数据模型核心

- `Users`: 用户基础信息、学习偏好、进度统计
- `Scenarios`: 场景定义、难度分级、训练目标
- `Sessions`: 训练会话记录、用户输入、AI反馈、评分
- `Examples`: 优质范例库，结构化存储领域最佳实践

### 关键流程

**场景训练流程**: 用户选择场景 → AI生成情景问题 → 用户输入回答 → AI多维度评估（内容/结构/语气/用词）→ 提供改进方案和范例对比 → 用户重练 → 进度保存。

### 扩展性设计

预留企业批量导入API，支持B2B企业培训客户接入；场景化模块可插拔，便于快速扩展新训练领域。
