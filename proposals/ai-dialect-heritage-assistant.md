# AI 方言传承助手 - 让奶奶的声音在家族中延续

> **一句话卖点**：孙辈在除夕夜用手机录下奶奶的方言祝福，AI 自动生成带拼音的方言卡片和表情包，让乡音在家族中代代相传

## 概述

基于 Issue #91 创建的详细设计文档。

**核心问题**：中国有 2.6 亿银发族，他们的方言承载着家族记忆和文化根脉，但缺乏工具让年轻人愿意学习、让老人轻松传承。现有方案要么是学术词典（枯燥），要么是短视频（碎片化），都缺乏**情感连接**。

**解决方案**：通过 AI 语音识别 + 多模态内容生成，把方言学习变成家庭互动游戏，让老人录制、年轻人编辑、孩子玩耍，三代人共同完成家族方言词典。

## 功能设计

### 核心功能

1. **🎤 家庭语音采集**
   - 老人录制日常用语（支持情绪标记：责备、撒娇、讲故事）
   - 自动生成拼音和声调标注
   - 支持方言变体识别（同一句话的不同说法）

2. **🎨 情感化内容生成**
   - 方言表情包（奶奶的口头禅 + AI 插图）
   - 家族方言小故事（配上场景插图）
   - 生日/节日祝福视频（方言配音 + 字幕）

3. **🎮 游戏化学习**
   - 「猜猜奶奶说什么」互动小游戏
   - 方言等级徽章系统（入门 → 流利 → 母语级）
   - 家庭方言挑战赛排行榜

4. **📖 家族方言词典**
   - 自动生成专属「我家的方言」数字小册子
   - 按场景分类（吃饭、睡觉、吵架、撒娇）
   - 支持导出为实体书（可打印）

5. **🤝 多代协作**
   - 老人录、年轻人编辑、孩子玩
   - 远程视频时自动推送「今日方言」
   - 家庭群分享功能

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| 方言 ASR 准确率低 | 采用众包模式，用户纠正后反馈训练模型 |
| 老人不会用复杂软件 | 极简界面 + 语音引导 + 子女代操作模式 |
| 使用频率低（仅节庆） | 「每日方言」推送 + 游戏化徽章维持粘性 |
| 隐私担忧（家庭录音） | 端到端加密 + 本地存储优先 + 明确数据政策 |
| 方言多样性难以覆盖 | 先聚焦 5-10 种主要方言，逐步扩展 |
