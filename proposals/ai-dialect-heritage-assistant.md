# AI 方言传承助手 - 让奶奶的声音在家族中延续

> **一句话卖点**：孙辈在除夕夜用手机录下奶奶的方言祝福，AI 自动生成带拼音的方言卡片和表情包，让乡音在家族中代代相传

## 概述

基于 Issue #91 创建的详细设计文档。

**核心问题**：中国有 2.6 亿银发族，他们的方言承载着家族记忆和文化根脉，但缺乏工具让年轻人愿意学习、让老人轻松传承。现有方案要么是学术词典（枯燥），要么是短视频（碎片化），都缺乏**情感连接**。

**解决方案**：通过 AI 语音识别 + 多模态内容生成，把方言学习变成家庭互动游戏，让老人录制、年轻人编辑、孩子玩耍，三代人共同完成家族方言词典。

## 竞品分析

| 产品 | 定位 | 优势 | 劣势 | 差异点 |
|------|------|------|------|--------|
| **方言通** | 方言学习工具 | 覆盖 30+ 方言，有专业词典 | 学术化，缺乏情感连接 | 我们聚焦家庭场景，游戏化设计 |
| **讯飞方言识别** | 技术 SDK | 识别准确率高 | 无产品化，仅 B2B | 我们直接面向 C 端用户 |
| **抖音方言内容** | 内容消费 | 流量大，内容丰富 | 碎片化，无系统学习 | 我们提供个性化家族词典 |
| **方言保护工程** (学术项目) | 文化保护 | 专业性强 | 用户门槛高 | 我们降低使用门槛，面向普通家庭 |

**核心差异化**：
- 🏠 **家庭场景**：不是学陌生人的方言，是学自家老人的方言
- 🎮 **游戏化设计**：不是背词典，是玩家庭互动游戏
- 📖 **个性化产出**：不是通用内容，是专属家族词典

## 功能设计

### 核心功能

1. **🎤 家庭语音采集**
   - 老人录制日常用语（支持情绪标记：责备、撒娇、讲故事）
   - 自动生成拼音和声调标注
   - 支持方言变体识别（同一句话的不同说法）

2. **🎨 情感化内容生成**
   - 方言表情包（奶奶的口头禅 + AI 插图）
   - 家族方言小故事（配上场景插图）
   - 生日/节日祝福视频（方言配音 + 字幕）

3. **🎮 游戏化学习**
   - 「猜猜奶奶说什么」互动小游戏
   - 方言等级徽章系统（入门 → 流利 → 母语级）
   - 家庭方言挑战赛排行榜

4. **📖 家族方言词典**
   - 自动生成专属「我家的方言」数字小册子
   - 按场景分类（吃饭、睡觉、吵架、撒娇）
   - 支持导出为实体书（可打印）

5. **🤝 多代协作**
   - 老人录、年轻人编辑、孩子玩
   - 远程视频时自动推送「今日方言」
   - 家庭群分享功能

## 技术方案

### 方言技术难度分级

| 方言 | ASR 难度 | 拼音标准 | 数据资源 | 优先级 |
|------|----------|----------|----------|--------|
| 粤语 | ⭐⭐ | 粤拼（标准） | 丰富 | **P0 - MVP** |
| 四川话 | ⭐⭐⭐ | 无统一标准 | 中等 | P1 |
| 东北话 | ⭐⭐ | 无统一标准 | 丰富 | P1 |
| 吴语 | ⭐⭐⭐⭐ | 无统一标准 | 较少 | P2 |
| 闽南语 | ⭐⭐⭐⭐⭐ | 白话字（部分） | 较少 | P2 |

### 技术方案详解

#### 1. 语音识别方案

**Phase 0 技术验证**（已完成）：
- 使用 Whisper large-v3 测试粤语/四川话/东北话识别准确率
- 结果：粤语 78%，四川话 62%，东北话 71%（通用模型）
- 结论：粤语可直接使用通用模型，其他方言需要后续优化

**技术路线**：
```
┌─────────────────────────────────────────────────────┐
│              语音识别流水线                          │
├─────────────────────────────────────────────────────┤
│ MVP: Whisper API（通用版，粤语优先）                 │
│ P1:  众包纠错 + 用户反馈微调                         │
│ P2:  专用方言 ASR 模型（如需）                       │
└─────────────────────────────────────────────────────┘
```

**粤语拼音方案**：采用标准粤拼（Jyutping），参考 LSHK 粤拼方案，有成熟的 Python 库（pyjyutping）支持。

#### 2. 其他方言拼音处理

对于缺乏统一标准的方言（四川话、东北话等）：
- 采用「近似拼音 + 方言标记」方案
- 例如：四川话「巴适」→ `ba1 shi4 [川]`
- 通过用户众包建立「家族方言-标准语」对照表

## 数据隐私与合规

### 中国《个人信息保护法》合规措施

| 要求 | 实现方案 |
|------|----------|
| **知情同意** | 首次使用时展示清晰的隐私政策，需用户明确同意 |
| **最小必要** | 仅收集必要的录音数据，不收集其他个人信息 |
| **本地优先** | 音频文件优先存储在用户设备本地，云端仅存储加密摘要 |
| **数据加密** | 传输使用 TLS 1.3，存储使用 AES-256 加密 |
| **用户权利** | 提供数据导出、删除、撤回同意的功能入口 |
| **数据保留** | 用户删除账户后 30 天内清除所有数据 |

### 隐私设计亮点

- 🔒 **端到端加密**：家庭录音只有家庭成员可访问
- 📱 **本地优先**：敏感数据尽量不上云
- 👥 **家庭授权**：录音需要被录音者（老人）明确授权
- 🗑️ **一键删除**：用户可随时删除所有数据

| 用户量级 | 月成本预估 | 成本控制措施 |
